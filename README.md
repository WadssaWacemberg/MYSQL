Projetos de Banco de Dados - MySQL
Este repositório contém as atividades práticas desenvolvidas para o aprendizado de Banco de Dados Relacional utilizando MySQL.

📂 Conteúdo do Repositório
O repositório está organizado em três scripts principais, conforme solicitado:

Atividade 01 - RH: Gerenciamento de dados de colaboradores, incluindo filtros salariais e atualizações cadastrais.

Atividade 02 - E-commerce: Controle de estoque e precificação de produtos com filtros de valor.

Atividade 03 - Registro Escolar: Gestão de notas e informações de estudantes utilizando conceitos de abstração.

🛠️ Tecnologias Utilizadas
MySQL: Linguagem de consulta estruturada (SQL) para criação e manipulação de dados .

GitHub: Versionamento e hospedagem de código.

🚀 Como Executar
Para rodar os scripts, basta copiar o conteúdo de cada arquivo .sql e executá-lo em seu ambiente MySQL (como o MySQL Workbench ou terminal).
